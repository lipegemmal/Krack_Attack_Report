\section{Introdução}
	O Krack Attack é um ataque ao Wifi, mais precisamente no protocolo WPA2 (\textit{Wi-Fi Protected Access}) responsável pela segurança. Mesmo com a prova formal da segurança de algumas partes no WPA2 não foi possível prever a falha ao se usar o Man In The Middle para bloquear a recepção de um pacote se confirmação no protocolo de conexão 4-Way Handshake do WPA2. 
	Uma situação específica também pode ocorrer com os dispositivos Android e Linux que reseta a chave de  autenticação para um valor fixo conhecido, assim facilitando ainda mais o acesso aos dados do dispositivo.
